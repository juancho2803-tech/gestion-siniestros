Crear usuario

Definí las etapas del proceso, en el orden en que aparecen en la hoja de ruta del trackeo.

Importar órdenes desde Excel/CSV

Se crean/actualizan las órdenes con Tipología EXTERNA CHAPA / PINTURA y Estado Abierta. Cliente, dominio y modelo se cargan solos; compañía de seguro, perito, N° de siniestro y detalle quedan para completar a mano. Si una orden ya existe (mismo N° de orden), se actualizan sus datos sin tocar la etapa ni lo que ya cargó el asesor. Además, cualquier caso que ya tenías cargado y que en este archivo aparece facturado o cerrado en el DMS se cierra solo como Entregado — así la base siempre refleja los casos realmente abiertos.

Verificar una orden puntual

Después de "Analizar archivo", buscá acá un N° de orden para ver exactamente qué valor trae la celda de fecha (crudo, sin interpretar) y qué fecha calculó el sistema — útil para diagnosticar discrepancias puntuales.

Re-vincular asesores

Si creaste o corregiste un usuario después de importar, los casos que quedaron "sin usuario" no se actualizan solos. Tocá esto para volver a cruzar por nombre todos los casos sin asesor asignado, sin necesidad de reimportar el archivo.

Detectar duplicados

Busca casos con el mismo dominio abiertos el mismo día (típicamente franquicia + siniestro cargados por error como dos casos). Desde acá no se borra nada solo — entrá al caso puntual para eliminarlo.

Zona de riesgo

Borra TODOS los siniestros, su historial, novedades y auditoría — para dejar la base en cero antes de lanzar el sistema. Los usuarios y las etapas configuradas NO se tocan.
